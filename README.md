# 🛒 ShoppyGlobe API

ShoppyGlobe is a full-stack e-commerce backend project built using Node.js, Express.js, and MongoDB. It provides RESTful APIs for managing users, products, and shopping cart functionality.

## 🚀 Features
- User authentication (Register/Login)
- Product management
- Cart functionality
- RESTful API structure
- Middleware-based architecture

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## 📁 Project Structure
- routes/ → API routes
- models/ → Database schemas
- middleware/ → Custom middlewares
- config/ → Database configuration
